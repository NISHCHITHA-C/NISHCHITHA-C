### ✨ Creative Portfolio
A single-page, animated portfolio lives in this repo — `index.html`, `styles.css`, `script.js`.
Enable **GitHub Pages** (Settings → Pages → deploy from `main`, root) to publish it.

Features: animated data-flow canvas background, typewriter hero, self-building ETL
terminal, pipeline view (Ingest → Transform → Orchestrate → Serve), experience
timeline, project cards, animated skill bars, and count-up stats. No build step,
no dependencies.

Run locally:
```bash
python3 -m http.server 8000   # open http://localhost:8000
```
